# Year 5 Mathematics — Term 2 Syllabus

**Teacher:** Ms. Sarah Chen
**School:** EduX Primary School
**Curriculum:** Australian Curriculum v9.0 — Mathematics

---

## Week 1–2: Place Value and Number Patterns (AC9M5N01)

### Learning Outcomes
- Understand place value of whole numbers up to millions
- Identify and describe number patterns including multiples and factors
- Round whole numbers to the nearest 10, 100, and 1,000

### Key Activities
- Place value chart activities using base-10 blocks
- Number pattern investigation: finding rules for growing patterns
- Real-world rounding: estimating costs at a pretend shop

### Assessment
- In-class quiz on place value (Week 2, Wednesday)

---

## Week 3–4: Fractions — Concepts and Equivalence (AC9M5N03)

### Learning Outcomes
- Represent fractions on number lines and area models
- Identify and generate equivalent fractions
- Compare and order fractions with related denominators
- Simplify fractions to their lowest terms

### Key Activities
- Fraction wall exploration: building fraction walls with paper strips
- Pizza sharing activity: dividing pizzas into equal parts to discover equivalence
- Fraction card game: matching equivalent fractions

### Assessment
- Fraction equivalence worksheet (Week 4, Friday)

---

## Week 5–6: Fractions — Operations (AC9M5N04)

### Learning Outcomes
- Add and subtract fractions with the same denominator
- Add and subtract fractions with related denominators (e.g., halves and quarters)
- Divide whole numbers by unit fractions using visual models
- Solve word problems involving fractions in everyday contexts

### Key Activities
- Fraction addition using fraction strips and number lines
- Pizza division challenge: if 3/4 of a pizza is shared among 3 people, how much does each get?
- Real-world fraction problems: recipes, measurements, sharing equally
- Fraction division with visual models (area models and number lines)

### Assessment
- Fraction operations test (Week 6, Thursday)
- Homework: fraction word problems worksheet

---

## Week 7–8: Decimals and Percentages (AC9M5N06)

### Learning Outcomes
- Understand the relationship between fractions, decimals, and percentages
- Read and write decimals to thousandths
- Compare and order decimals
- Convert between simple fractions, decimals, and percentages

### Key Activities
- Decimal number line activities
- Shopping maths: calculating discounts using percentages
- Fraction-decimal-percentage matching game

### Assessment
- Decimals and percentages quiz (Week 8, Tuesday)

---

## Week 9–10: Data and Statistics (AC9M5ST01)

### Learning Outcomes
- Collect, organise, and display data using tables and graphs
- Interpret data displays including column graphs and dot plots
- Calculate the mean (average) of a data set
- Draw conclusions from data and make predictions

### Key Activities
- Class survey project: design, collect, and display data
- Weather data analysis: interpreting real Bureau of Meteorology data
- Mean calculation with hands-on objects (stacking cubes evenly)

### Assessment
- Data project presentation (Week 10)
- End-of-term mathematics assessment (Week 10, Friday)
